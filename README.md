*** Aplikacion Web për Katalog & Autentikim**

***Përshkrimi**

Ky projekt është një aplikacion web i zhvilluar në PHP që mundëson autentikimin e përdoruesve dhe shfletimin e një katalogu modern me UI/UX të avancuar.

Sistemi është ndërtuar me strukturë të mirë të faqeve dhe logjikë funksionale, duke përdorur ndarje të qartë midis frontend dhe backend.

Aplikacioni mbështet këto role përdoruesish:

- User (përdorues normal)
- (strukturë e gatshme për zgjerim në admin)

---

*** Funksionalitetet**

- Login dhe register i përdoruesve
- Logout dhe menaxhim i sesioneve
- Menaxhim i gjendjes së përdoruesit (session)
- Implementim i klasës `User` (OOP)
- Shfaqje e katalogut në grid layout
- Modal për detaje të elementeve
- Header dhe filter për katalog
- Hero section modern (UI)
- Validim i inputeve (bazik)
- Strukturim modular i CSS dhe JS

---

*** Teknologjitë e përdorura**

- PHP
- HTML
- CSS
- JavaScript
- Git & GitHub

---

***Konceptet e implementuara**

- Variabla, kushte dhe cikle
- Funksione
- Programim i orientuar në objekte (OOP)
- Klasa dhe objekte
- Enkapsulim
- Sessions dhe Cookies
- Validim i inputeve

---

*** Struktura e projektit**

- `index.php` – Faqja kryesore
- `auth/login.php` – Login
- `auth/register.php` – Register
- `auth/logout.php` – Logout
- `class/user.php` – Klasa User (logjika backend)
- `assets/css/auth.css` – Styling për autentikim
- `assets/css/catalog.css` – Styling për katalog
- `assets/css/hero-section.css` – Styling për hero section
- `assets/js/catalog.js` – Logjika e katalogut

---

*** Ekzekutimi i projektit**

1. Instaloni XAMPP ose WAMP
2. Vendosni projektin në folderin `htdocs`
3. Startoni Apache
4. Hapni në browser
